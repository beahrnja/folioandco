# 📚 Folio & Co.

## Quiet rooms, loud books.

A front-end web project and brand identity for Folio & Co., a fictional independent bookshop with a carefully curated inventory — independent fiction, translated novels, poetry you can hold in one hand, and the odd cookbook they couldn't say no to.
Built as a portfolio project to practice cohesive brand design and semantic, accessible HTML/CSS.

### 🛠️ Built With

- HTML5 — semantic, accessible markup
- CSS3 — custom properties, Flexbox/Grid, responsive layout
- Vanilla JavaScript — lightweight interactions, no frameworks


### ✨ Features

- Brand identity system — typography, color palette, voice, and visual language developed from scratch to match the shop's quiet-but-opinionated personality
- Responsive design — layout adapts cleanly across mobile, tablet, and desktop
- Accessibility-first — semantic HTML, proper heading hierarchy, keyboard navigability, and sufficient color contrast throughout


### 🎨 Design Decisions
Folio & Co. is built around a tension: a hushed, intimate physical space that stocks books loud enough to change your week. The brand leans into that contrast — restrained visuals, deliberate whitespace, and copy that has a point of view.
The design system prioritizes:

- Legibility over decoration — typography does the heavy lifting
- Warmth without clutter — a limited palette and generous spacing keep things calm
- Voice consistency — dry, a little stubborn, quietly enthusiastic about good books
